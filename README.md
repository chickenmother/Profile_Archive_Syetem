# Profile_Archive_Syetem

目標アプリ：プロフィールアーカイブシステム
	概要：チームリーダーがプロジェクトへ参加する適切な人材を選定するために、社員が持つスキルや過去に携わったプロジェクトを整理する。リーダーはフィルター機能、プロフィール内のコメント機能などを使ってプロジェクトにマッチした者を選別する。また社員は自身のアカウントにログインし、プロフィールを編集することができ、過去に共に仕事をした同僚のプロフィール内でコメントを残す、編集、削除ことができる。
